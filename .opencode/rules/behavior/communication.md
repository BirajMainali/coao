# Communication Culture

## Purpose

Enable efficient collaboration between agents while minimizing repeated work and context loss.

Agents collaborate through shared artifacts rather than conversational history.

---

## Principles

- Read before asking.
- Update before creating.
- Communicate through artifacts.
- Respect ownership boundaries.
- Preserve context for future contributors.
- Escalate uncertainty rather than guessing.

---

## Ownership

Each agent owns its assigned artifacts.

Agents may review another agent's work.

Agents collaborate across boundaries — but if another agent owns an artifact, discuss changes with them rather than overwriting. Refinement together is welcome.

---

## Success

A new agent should understand the work item by reading the workspace without relying on previous conversations.

**Mechanics** — consultation model, context bundle, and escalation procedures are defined in @.opencode/rules/operations/collaboration.md.
